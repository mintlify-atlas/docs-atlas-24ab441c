---
title: "Create Session"
api: "POST /api/sessions"
description: "Create a new Reddit-based session with AI-generated captions"
---

## Authentication

This endpoint requires authentication and credits. Sessions creation requires the `session_create` feature credit.

## Request Body

<ParamField body="title" type="string" required>
  Session title (minimum 1 character)
</ParamField>

<ParamField body="subreddits" type="string" required>
  Comma-separated list of subreddit names (e.g., "joi,femdom,gonewild")
</ParamField>

<ParamField body="categoryId" type="integer" optional>
  Category ID to assign to this session
</ParamField>

<ParamField body="intervalMin" type="integer" default={3}>
  Minimum slide interval in seconds (1-60)
</ParamField>

<ParamField body="intervalMax" type="integer" default={10}>
  Maximum slide interval in seconds (1-60)
</ParamField>

<ParamField body="transition" type="string" default="fade">
  Transition effect: "fade", "slide", "zoom", "flip", or "none"
</ParamField>

<ParamField body="aiPrompt" type="string" optional>
  Custom AI prompt for caption generation
</ParamField>

<ParamField body="captionTheme" type="string" optional>
  Caption theme: "joi", "forced-bi", "beta", "cbt", "cuckold", or "custom"
</ParamField>

<ParamField body="thumbnail" type="string" optional>
  Custom thumbnail URL. If not provided, auto-generated from subreddits
</ParamField>

<ParamField body="isPublic" type="boolean" default={false}>
  Whether the session is publicly accessible
</ParamField>

<ParamField body="shareToCommunity" type="boolean" default={false}>
  Whether to share session to the community feed
</ParamField>

<ParamField body="isFavorite" type="boolean" default={false}>
  Whether to mark as favorite
</ParamField>

<ParamField body="isManualMode" type="boolean" default={false}>
  Whether to disable auto-advance in the player
</ParamField>

## Response

Returns the created session object with an automatically assigned ID.

<ResponseField name="id" type="integer">
  Unique session identifier
</ResponseField>

<ResponseField name="title" type="string">
  Session title
</ResponseField>

<ResponseField name="userId" type="string">
  Owner's user ID
</ResponseField>

<ResponseField name="subreddits" type="array">
  Normalized array of subreddit names
</ResponseField>

<ResponseField name="categoryId" type="integer">
  Assigned category ID
</ResponseField>

<ResponseField name="categorySource" type="string">
  "manual" if explicitly set, "default" if auto-assigned
</ResponseField>

<ResponseField name="thumbnail" type="string" optional>
  URL to auto-generated or custom thumbnail
</ResponseField>

<ResponseField name="createdAt" type="timestamp">
  Session creation timestamp
</ResponseField>

<RequestExample>
```bash cURL

  -H "Content-Type: application/json" \
  -H "Cookie: connect.sid=YOUR_SESSION_COOKIE" \
  -d '{
    "title": "My JOI Session",
    "subreddits": "joi,femdom",
    "intervalMin": 5,
    "intervalMax": 15,
    "transition": "fade",
    "captionTheme": "joi",

```
</RequestExample>

## Notes

- The server automatically pre-populates the session with up to 25 media items in the background
- Subreddits are normalized (lowercased, `r/` prefix removed)
- If `categoryId` is not provided, the session is assigned to the "Uncategorized" category
- Credits are deducted after successful creation
- Activity is tracked for analytics
